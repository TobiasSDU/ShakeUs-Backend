Party Endpoints

POST /party/create

                
                    {
                        "activityPackId": string,
                        "hostName": string
                    }

Request Body

                
                    {
                        "hostId": string,
                        "partyId": string
                    }

Response Body (Error Code: 400)

                    {
                        "partyId": string,

                    {
                        "_id": string,
                        "hosts": string[],
                        "primaryHost": string,
                        "guests": string[],
                        "activityPackId": string
                    }

Response Body (Error Code: 400)
