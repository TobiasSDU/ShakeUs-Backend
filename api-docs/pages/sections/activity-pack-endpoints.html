Activity Pack Endpoints

POST

/activity-pack/create

Reponse Codes:

200

|

400

Description

Creates a new activity pack.

title: The title of the activity pack.
description: A description explaining the content of the activity pack.

Socket.IO Event

No event emitted 

                    
                        // Request Body
                        {
                            "title": string,
                            "description": string
                        }
                    
                
                    
                        // Response Body
                        {
                            "activityPackId": string
                        }

GET

/activity-pack/show

Reponse Codes:

200

|

400

Description

Returns information about the activity pack matching the provided id.

activityPackId: The id of the activity pack you want information about.

Socket.IO Event

No event emitted 

                    
                        // Request Body
                        {
                            "activityPackId": string
                        }
                    
                
                    
                        // Response Body
                        {
                            "_id": string,
                            "title": string,
                            "description": string,
                            "activities": string[],
                        }

PATCH

/activity-pack/title/update

Reponse Codes:

200

|

400

Description

Updates the title of an activity pack.

activityPackId: The activity pack which should be updated.
newTitle: The activity pack's new title

Socket.IO Event

Event name: activity-pack-title-updated
Data: { removedActivityId, updatedField, message } 

                    
                        // Request Body
                        {
                            "activityPackId": string,
                            "newTitle": string
                        }
                    
                
                    
                        // Response Body
                        HTTP: 200 OK

|

                        }
                    
                
                    
                        // Response Body
