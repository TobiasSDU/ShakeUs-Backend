Guest Endpoints

GET

/guest

Reponse Codes:

200

|

400

Description

Returns information about the guest matching the provided id.

guestId: The id of the guest you want information about.

Socket.IO Event

No event emitted 

                    
                        // Request Body
                        {
                            "guestId": string,
                        }
                    
                
                    
                        // Response Body
                        {
                            "_id": string,
                            "name": string
                        }

PATCH

/guest

Reponse Codes:

200

|

400

Description

Updates the name of a guest.

guestId: The guest which should be updated.
newName: The guest's new name.

Socket.IO Event

Event name: username-updated
Data: { user, message } 

                    
                        // Request Body
                        {
                            "guestId": string,
                            "newName": string
                        }
                    
                
                    
                        // Response Code
                        HTTP: 200 OK
